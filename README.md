## CLI usage

install using npm `npm i -g @kkito/drumstick`

### start server 

`drumstick-server`

or append some options 

`drumstick-server port=5432 secret=aaaa`

### client in cli

`drumstick-client http://www.google.com`

`drumstick-client http://www.baidu.com port=5432 secret=aaaa`
